## Quick Install

The fastest way to get started is with our CLI:

```bash
npx create-ink-agent my-agent
```

This scaffolds a complete project with all dependencies configured.

## Manual Installation

For existing projects or custom setups:

<Steps>
  <Step title="Install core packages">
    ```bash
    npm install @convex-agent/core convex
    ```
  </Step>

  <Step title="Initialize Convex">
    ```bash
    npx convex init
    ```

    This creates the `convex/` directory for your backend functions.
  </Step>

  <Step title="Add INK schema">
    Create `convex/schema.ts`:

    ```typescript
    import { defineSchema, defineTable } from "convex/server";
    import { v } from "convex/values";

    export default defineSchema({
      // Agent state (singleton)
      state: defineTable({
        mood: v.string(),
        health: v.number(),
        routine: v.string(),
        volatility: v.number(),
        summary: v.string(),
        updatedAt: v.number(),
      }),

      // Activity logs
      logs: defineTable({
        mood: v.string(),
        health: v.number(),
        routine: v.string(),
        summary: v.string(),
        detail: v.string(),
        timestamp: v.number(),
      }).index("by_timestamp", ["timestamp"]),

      // Memory system
      memories: defineTable({
        type: v.union(
          v.literal("episodic"),
          v.literal("semantic"),
          v.literal("emotional"),
          v.literal("procedural"),
          v.literal("reflection")
        ),
        content: v.string(),
        embedding: v.array(v.float64()),
        importance: v.number(),
        salience: v.number(),
        confidence: v.number(),
        sentiment: v.object({
          score: v.number(),
          label: v.string(),
        }),
        metadata: v.any(),
        createdAt: v.number(),
        lastAccessedAt: v.number(),
      })
        .index("by_type", ["type"])
        .index("by_importance", ["importance"])
        .vectorIndex("by_embedding", {
          vectorField: "embedding",
          dimensions: 768,
        }),
    });
    ```
  </Step>

  <Step title="Create agent configuration">
    Create `convex/agent.ts`:

    ```typescript
    import { Agent } from "@convex-agent/core";

    export const agent = new Agent({
      name: "my-agent",
      description: "A helpful autonomous agent",
      model: "anthropic/claude-3.5-sonnet",
      tickInterval: 300000, // 5 minutes
    });
    ```
  </Step>
</Steps>

## Environment Variables

Create `.env.local` with the following:

```bash
# Required
NEXT_PUBLIC_CONVEX_URL=your-convex-deployment-url
OPENROUTER_KEY=your-openrouter-api-key

# Optional but recommended
GEMINI_API_KEY=your-gemini-key  # For embeddings
TICK_TOKEN=secure-random-token  # Auth for tick endpoint
WRITE_TOKEN=secure-random-token # Auth for write operations
```

### Getting API Keys

<AccordionGroup>
  <Accordion title="Convex URL">
    Run `npx convex dev` and copy the deployment URL from the output.
  </Accordion>

  <Accordion title="OpenRouter Key">
    1. Go to [openrouter.ai](https://openrouter.ai)
    2. Sign up for a free account
    3. Navigate to API Keys and create a new key
  </Accordion>

  <Accordion title="Gemini API Key">
    1. Go to [Google AI Studio](https://aistudio.google.com)
    2. Create a new API key
    3. This is used for generating embeddings (768 dimensions)
  </Accordion>
</AccordionGroup>

## Project Structure

After installation, your project will look like:

```
my-agent/
├── app/                    # Next.js frontend (optional)
│   ├── page.tsx
│   └── layout.tsx
├── convex/                 # Convex backend
│   ├── _generated/         # Auto-generated (don't edit)
│   ├── schema.ts           # Database schema
│   ├── agent.ts            # Agent configuration
│   ├── tick.ts             # Tick execution
│   ├── http.ts             # HTTP endpoints
│   └── memory/             # Memory system
│       ├── index.ts
│       ├── embeddings.ts
│       └── retrieval.ts
├── plugins/                # Custom plugins
├── .env.local              # Environment variables
├── convex.json             # Convex configuration
└── package.json
```

## Verify Installation

Start the development server:

```bash
npm run dev
```

Check that everything is working:

```bash
# Health check
curl http://localhost:3001/health

# Should return:
# {"status":"ok","model":"anthropic/claude-3.5-sonnet","lastTick":null}
```

## Docker Installation

For containerized deployments:

```dockerfile
FROM node:18-alpine

WORKDIR /app
COPY package*.json ./
RUN npm ci

COPY . .
RUN npm run build

ENV NODE_ENV=production
CMD ["npm", "start"]
```

```yaml
# docker-compose.yml
version: '3.8'
services:
  agent:
    build: .
    ports:
      - "3000:3000"
    environment:
      - NEXT_PUBLIC_CONVEX_URL=${CONVEX_URL}
      - OPENROUTER_KEY=${OPENROUTER_KEY}
```
