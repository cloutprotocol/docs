## Social Plugins

### Twitter Plugin

Post tweets, reply to mentions, monitor keywords. Uses mood-gating to post only when expressive.

| Config | Type | Description |
|--------|------|-------------|
| `apiKey` | string | Twitter API key |
| `apiSecret` | string | Twitter API secret |
| `accessToken` | string | OAuth access token |
| `accessSecret` | string | OAuth access secret |
| `postEnabled` | boolean | Enable posting |
| `cooldownMinutes` | number | Minutes between posts |

**Actions:** `post(content)`, `getMentions()`

**Reference:** `plugins/twitter.ts`

### Discord Plugin

Community engagement through Discord bot integration.

| Config | Type | Description |
|--------|------|-------------|
| `token` | string | Bot token |
| `guildId` | string | Server ID |
| `channelId` | string | Default channel |
| `respondToMentions` | boolean | Auto-respond |

**Actions:** `sendMessage(content, channelId?)`

**Reference:** `plugins/discord.ts`

## Data Plugins

### News Plugin

Multi-source news aggregation from RSS feeds and APIs.

| Config | Type | Description |
|--------|------|-------------|
| `sources` | NewsSource[] | Feed URLs and types |
| `refreshMinutes` | number | Fetch interval |
| `keywords` | string[] | Filter terms |

**Actions:** `getLatest(limit?)`

**Reference:** `plugins/news.ts`

### Markets Plugin

Real-time cryptocurrency price data from CoinGecko.

| Config | Type | Description |
|--------|------|-------------|
| `symbols` | string[] | Coin IDs (e.g., "bitcoin") |
| `refreshMinutes` | number | Cache refresh interval |

**Actions:** `getPrice(symbol)`, `getAllPrices()`

**Reference:** `plugins/markets.ts`

## Using Example Plugins

```typescript
import { Agent } from "./src";
import { ConvexBackend } from "./src/backend";
import { createTwitterPlugin } from "./plugins/twitter";
import { createNewsPlugin } from "./plugins/news";

const agent = new Agent({
  backend: new ConvexBackend({ url: process.env.CONVEX_URL! }),
  plugins: [
    createTwitterPlugin({
      apiKey: process.env.TWITTER_API_KEY!,
      apiSecret: process.env.TWITTER_API_SECRET!,
      accessToken: process.env.TWITTER_ACCESS_TOKEN!,
      accessSecret: process.env.TWITTER_ACCESS_SECRET!,
      postEnabled: true,
      cooldownMinutes: 60
    }),
    createNewsPlugin({
      sources: [{ name: "coindesk", url: "https://coindesk.com/rss" }],
      refreshMinutes: 15
    })
  ]
});

// Execute action
await agent.executeAction("twitter", "post", { content: "Hello!" });
```
