## Social Plugins

### @clarkos/plugin-twitter

Full Twitter integration for posting, replying, and monitoring.

```bash
npm install @clarkos/plugin-twitter
```

```typescript
import { twitterPlugin } from "@clarkos/plugin-twitter";

const agent = new Agent({
  plugins: [
    twitterPlugin({
      // Required credentials
      apiKey: process.env.TWITTER_API_KEY,
      apiSecret: process.env.TWITTER_API_SECRET,
      accessToken: process.env.TWITTER_ACCESS_TOKEN,
      accessSecret: process.env.TWITTER_ACCESS_SECRET,

      // Behavior settings
      postEnabled: true,
      replyEnabled: true,
      minHealthToPost: 50,
      cooldownMinutes: 30,

      // Content filters
      maxLength: 280,
      bannedWords: ["shill", "guaranteed"],
    }),
  ],
});
```

**Features:**
- Post tweets from agent thoughts
- Reply to mentions
- Monitor keywords
- Thread support
- Media attachments

**Actions:**
- `twitter.post(content)` - Post a tweet
- `twitter.reply(tweetId, content)` - Reply to a tweet
- `twitter.like(tweetId)` - Like a tweet
- `twitter.getMentions()` - Get recent mentions

**Features:**

- Reaction handling

## Data Plugins

### @clarkos/plugin-news

Multi-source news aggregation with smart filtering.

```bash
npm install @clarkos/plugin-news
```

```typescript
import { newsPlugin } from "@clarkos/plugin-news";

const agent = new Agent({
  plugins: [
    newsPlugin({
      sources: [
        { name: "coindesk", url: "https://coindesk.com/rss" },
        { name: "techcrunch", url: "https://techcrunch.com/feed" },
        { name: "hackernews", type: "hackernews", minScore: 100 },
      ],

      // Filtering
      keywords: ["ethereum", "ai", "crypto"],
      excludeKeywords: ["sponsored", "advertisement"],

      // Schedule
      refreshIntervalMinutes: 15,
      maxItemsPerSource: 10,

      // Processing
      generateEmbeddings: true,
      extractEntities: true,
    }),
  ],
});
```

**Features:**
- RSS/Atom feed support
- Hacker News integration
- Reddit subreddit monitoring
- Keyword filtering
- Automatic embedding generation
- Entity extraction

- Historical data
- Market sentiment

## Action Plugins

### @clarkos/plugin-trading

Paper and live trading capabilities.

```bash
npm install @clarkos/plugin-trading
```

```typescript
import { tradingPlugin } from "@clarkos/plugin-trading";

const agent = new Agent({
  plugins: [
    tradingPlugin({
      // Paper trading (safe)
      mode: "paper",
      initialBalance: 10000,

      // Or live trading (be careful!)
      // mode: "live",
      // exchange: "binance",
      // apiKey: process.env.BINANCE_API_KEY,
      // apiSecret: process.env.BINANCE_API_SECRET,

      // Risk management
      maxPositionSize: 0.1,  // 10% of portfolio
      stopLossPercent: 5,
      takeProfitPercent: 15,

      // Behavior
      autoTrade: false,  // Require manual confirmation
      requireConfidence: 0.8,
    }),
  ],
});
```

**Actions:**
- `trading.buy(symbol, amount, options)`
- `trading.sell(symbol, amount, options)`
- `trading.getPortfolio()`
- `trading.getHistory()`
- `trading.cancelOrder(orderId)`

## Utility Plugins

### @clarkos/plugin-scheduler

Advanced scheduling beyond crons.

```bash
npm install @clarkos/plugin-scheduler
```

```typescript
import { schedulerPlugin } from "@clarkos/plugin-scheduler";

const agent = new Agent({
  plugins: [
    schedulerPlugin({
      tasks: [
        {
          name: "morning-briefing",
          schedule: "0 9 * * *",  // 9 AM daily
          action: async (ctx) => {
            await ctx.action("twitter.post", "Good morning!");
          },
        },
        {
          name: "market-update",
          schedule: "0 */4 * * *",  // Every 4 hours
          action: async (ctx) => {
            const markets = await ctx.action("markets.getSummary");
            // Process and post...
          },
        },
      ],
    }),
  ],
});
```

### @clarkos/plugin-webhooks

Receive external webhooks.

```bash
npm install @clarkos/plugin-webhooks
```

```typescript
import { webhooksPlugin } from "@clarkos/plugin-webhooks";

const agent = new Agent({
  plugins: [
    webhooksPlugin({
      secret: process.env.WEBHOOK_SECRET,
      handlers: {
        "github/push": async (payload, ctx) => {
          await ctx.remember({
            type: "episodic",
            content: `New commit: ${payload.commits[0].message}`,
          });
        },
        "stripe/payment": async (payload, ctx) => {
          // Handle payment events
        },
      },
    }),
  ],
});
```

## Installation

All official plugins follow the same pattern:

```bash
# Install the plugin
npm install @clarkos/plugin-NAME

# Import and configure
import { NAMEPlugin } from "@clarkos/plugin-NAME";

const agent = new Agent({
  plugins: [NAMEPlugin({ ...config })],
});
```
