## Prerequisites

Before you begin, make sure you have:

- **Node.js 18+** installed
- **npm** or **pnpm** package manager
- An **OpenRouter API key** (or compatible LLM provider)

<Tip>
  Don't have an OpenRouter key? [Get one free](https://openrouter.ai) with $5 in credits.
</Tip>

## Create Your Agent

<Steps>
  <Step title="Clone the template">
    Clone the ClarkOS template repository:

    ```bash
    git clone https://github.com/clarkos/clarkos my-first-agent
    cd my-first-agent
    ```

    This gives you a fully-configured project with:
    - Convex backend setup
    - TypeScript configuration
    - Example plugins
    - Development scripts
  </Step>

  <Step title="Install dependencies">
    Install the required packages:

    ```bash
    npm install
    ```
  </Step>

  <Step title="Configure your environment">
    Set up your environment variables:

    ```bash
    cp .env.example .env.local
    ```

    Edit `.env.local` with your API keys:

    ```bash
    NEXT_PUBLIC_CONVEX_URL=your-convex-url
    OPENROUTER_KEY=your-openrouter-key
    TICK_TOKEN=your-tick-token
    ```
  </Step>

  <Step title="Start the development server">
    Launch your agent locally:

    ```bash
    npm run dev
    ```

    This runs both Convex and Next.js. Your agent is now running with:
    - Real-time state updates
    - HTTP API endpoints
    - Memory persistence
  </Step>
</Steps>

## Your First Tick

The **tick** is the heartbeat of your agent. Every tick, your agent:

1. Loads its current state and memories
2. Gathers context (news, knowledge, recent activity)
3. Processes through the LLM
4. Updates state and creates new memories
5. Optionally generates artifacts (tweets, images, etc.)

Trigger a manual tick:

```bash
curl -X POST http://localhost:3001/tick \
  -H "Authorization: Bearer your-tick-token"
```

Or watch it tick automatically based on your configured interval.

## Add a Plugin

Extend your agent with a plugin:

```typescript
// plugins/logger.ts
import type { Plugin } from "./src/plugins";

export const loggerPlugin: Plugin = {
  name: "logger",
  version: "1.0.0",
  description: "Logs tick activity",

  init(agent) {
    console.log("Logger plugin initialized");
  },

  onTick(context) {
    console.log(`Tick executed. Mood: ${context.state.mood}, Health: ${context.state.health}`);
    console.log(`Memories loaded: ${context.memories.length}`);
  }
};
```

Register it in your agent:

```typescript
import { Agent } from "./src";
import { ConvexBackend } from "./src/backend";
import { loggerPlugin } from "./plugins/logger";

const agent = new Agent({
  backend: new ConvexBackend({ url: process.env.CONVEX_URL! }),
  plugins: [loggerPlugin]
});

// Execute a tick
await agent.tick();
```

## Deploy to Production

When you're ready to go live:

<Steps>
  <Step title="Deploy Convex backend">
    ```bash
    npx convex deploy
    ```
  </Step>

  <Step title="Deploy frontend (optional)">
    Deploy to Vercel, Netlify, or any static host:

    ```bash
    npm run build
    vercel deploy
    ```
  </Step>

  <Step title="Set up scheduled ticks">
    Configure a cron job or use Convex's built-in scheduler:

    ```typescript
    // convex/crons.ts
    import { cronJobs } from "convex/server";

    const crons = cronJobs();
    crons.interval("agent tick", { minutes: 5 }, "tick:run");

    export default crons;
    ```
  </Step>
</Steps>
