## Prerequisites

Before you begin, make sure you have:

- **Node.js 18+** installed
- **npm** or **pnpm** package manager
- An **OpenRouter API key** (or compatible LLM provider)

<Tip>
  Don't have an OpenRouter key? [Get one free](https://openrouter.ai) with $5 in credits.
</Tip>

## Create Your Agent

<Steps>
  <Step title="Initialize the project">
    Run our CLI to scaffold a new agent project:

    ```bash
    npx create-ink-agent my-first-agent
    ```

    This creates a fully-configured project with:
    - Convex backend setup
    - TypeScript configuration
    - Example plugins
    - Development scripts
  </Step>

  <Step title="Configure your environment">
    Navigate to your project and set up your environment:

    ```bash
    cd my-first-agent
    cp .env.example .env.local
    ```

    Edit `.env.local` with your API keys:

    ```bash
    OPENROUTER_KEY=your-openrouter-key
    GEMINI_API_KEY=your-gemini-key  # For embeddings
    ```
  </Step>

  <Step title="Start the development server">
    Launch your agent locally:

    ```bash
    npm run dev
    ```

    Your agent is now running at `http://localhost:3000` with:
    - Real-time state updates
    - Automatic tick execution
    - Memory persistence
    - HTTP API endpoints
  </Step>
</Steps>

## Your First Tick

The **tick** is the heartbeat of your agent. Every tick, your agent:

1. Loads its current state and memories
2. Gathers context (news, knowledge, recent activity)
3. Processes through the LLM
4. Updates state and creates new memories
5. Optionally generates artifacts (tweets, images, etc.)

Trigger a manual tick:

```bash
curl -X POST http://localhost:3001/tick \
  -H "Authorization: Bearer your-tick-token"
```

Or watch it tick automatically based on your configured interval.

## Add a Plugin

Extend your agent with a plugin:

```typescript
// plugins/twitter.ts
import { Plugin } from "@convex-agent/core";

export const twitterPlugin: Plugin = {
  name: "twitter",
  version: "1.0.0",

  async onTick(context) {
    // Check if agent should tweet
    if (context.state.mood === "expressive") {
      const thought = context.getRecentThought();
      await this.postTweet(thought.content);
    }
  }
};
```

Register it in your agent:

```typescript
// convex/agent.ts
import { Agent } from "@convex-agent/core";
import { twitterPlugin } from "./plugins/twitter";

const agent = new Agent({
  name: "my-agent",
  plugins: [twitterPlugin]
});
```

## Deploy to Production

When you're ready to go live:

<Steps>
  <Step title="Deploy Convex backend">
    ```bash
    npx convex deploy
    ```
  </Step>

  <Step title="Deploy frontend (optional)">
    Deploy to Vercel, Netlify, or any static host:

    ```bash
    npm run build
    vercel deploy
    ```
  </Step>

  <Step title="Set up scheduled ticks">
    Configure a cron job or use Convex's built-in scheduler:

    ```typescript
    // convex/crons.ts
    import { cronJobs } from "convex/server";

    const crons = cronJobs();
    crons.interval("agent tick", { minutes: 5 }, "tick:run");

    export default crons;
    ```
  </Step>
</Steps>
